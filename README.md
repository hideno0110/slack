# README

## 概要
#generalチャンネルから過去１週間分のチャットを取得し、選択したものを削除する

## 実行手順
1. git cloneする
2. bundle install
3. slack tokenを環境変数に設定する
export SLACK_API_TOKEN='あなたのslack api token'
4. rails s
